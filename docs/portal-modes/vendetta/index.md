# Vendetta

HVT (High Value Target) mode. Each round one player is randomly designated as the HVT. Eliminating the HVT awards bonus score. The HVT receives an HP buff and a periodic ping system to even the odds.

- **Map:** Eastwood (`MP_Eastwood`)
- **Current version:** v0.3.0
- **Status:** Active development
- **Built from:** scratch in TypeScript
